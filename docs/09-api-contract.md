# 09. API Contract

Base URL (local): `http://localhost:8000`
Interactive docs (Swagger UI, auto-generated by FastAPI): `http://localhost:8000/docs`

All endpoints return JSON except `/session/{id}/pdf`, which streams a
`application/pdf` file.

## Health

`GET /health` ‚Üí `{"status": "ok"}`

## Sessions

`POST /session`
Creates a new project session.
Response: `{"session_id": "uuid"}`

## Upload

`POST /session/{session_id}/upload` (multipart/form-data, field `file`)
Loads the dataset and runs the Domain Specialist agent synchronously.
Response:
```json
{
  "dataset_name": "customers.csv",
  "n_rows": 400,
  "n_columns": 5,
  "columns": ["CustomerID", "Gender", "Age", "AnnualIncome", "SpendingScore"],
  "domain_brief": "### Likely domain\n..."
}
```

`GET /session/{session_id}/domain-brief` ‚Üí `{"domain_brief": "..."}`

## Reports

Each stage has a `POST` (compute + generate) and a matching cached `GET`.

| Stage key | POST route | Body |
|---|---|---|
| `data_health` | `/session/{id}/report/data_health` | none |
| `eda` | `/session/{id}/report/eda` | none |
| `clustering` | `/session/{id}/report/clustering` | `{"algorithm": "KMeans", "n_clusters": 5, "eps": 0.5, "min_samples": 5}` |
| `cluster_profiles` | `/session/{id}/report/cluster_profiles` | none |
| `cluster_comparison` | `/session/{id}/report/cluster_comparison` | `{"cluster_a": 0, "cluster_b": 1}` |
| `anomaly_detection` | `/session/{id}/report/anomaly_detection` | none |
| `recommendations` | `/session/{id}/report/recommendations` | none |
| `executive_dashboard` | `/session/{id}/report/executive_dashboard` | none |
| `executive_report` | `/session/{id}/report/executive_report` | none |

All report POST responses: `{"stage": "...", "markdown": "..."}`

`GET /session/{session_id}/report/{stage}` returns the same shape from
cache, or `404` if that stage hasn't been generated yet.

## PDF export

`GET /session/{session_id}/pdf?upto={stage}`
Returns a PDF (as a file download) containing every generated report
from the start of the pipeline up to and including `upto`.

## Error format
All errors: `{"detail": "human readable message"}` with an appropriate
HTTP status:
- `404` ‚Äî unknown session or stage not yet generated
- `409` ‚Äî stage prerequisites not met (e.g. clustering not run yet)
- `413` ‚Äî upload too large
- `503` ‚Äî LLM/search provider not configured (missing API key)
- `500` ‚Äî unexpected error
